# Kyberna vem do grego κυβερνάω (kybernáō), que significa pilotar, digirir, guiar, comandar, governar etc.
# Este nome veio à mente justamente porque é o "driven" de Data-Driven, além de variantes de Kyberna já terem precedentes na tecnologia, como o "Cibernético" e "Kubernetes".
# Por tanto, o projeto Kyberna visa justamente viabilizar que PMEs tenho acesso ao Data-Driven e Advanced Analytics, através de soluções padronizadas e centralizando custo visando viabilizar comercialmente o que antes era inviável.
